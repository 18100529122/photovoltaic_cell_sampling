# 光伏电池采样系统 - 任务计划

## 项目概述

基于 STM32G474VETx + FreeRTOS 的光伏电池多通道采样系统，实现26路模拟信号采集、数据处理、CAN通信及参数存储功能。

## 关键决策

1. **RTOS选择**: FreeRTOS (CMSIS-RTOS v2)
2. **MCU**: STM32G474VETx
3. **ADC方案**: 4个ADC+DMA，TIM2触发
4. **CAN方案**: FDCAN2，经典CAN格式，500Kbps波特率
5. **数据处理**: 冒泡排序，去掉10个最大/最小值，剩余180个求平均

## 错误记录

### 2026-07-02: ADC4通道映射重复问题
- **问题**: ADC4的Rank2和Rank3都映射到VS10，导致通道冲突
- **修复**:
  - Rank2-IN14-PE10 -> CUR10
  - Rank3-IN15-PE11 -> VS11
- **影响文件**: Core/App/data_process/data_process.c
