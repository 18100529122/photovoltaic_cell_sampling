# 引脚配置说明

## 电源和时钟引脚

| 引脚 | 功能 | 说明 |
|------|------|------|
| OSC32_IN | PC14 | 32.768kHz 晶振输入 |
| OSC32_OUT | PC15 | 32.768kHz 晶振输出 |
| OSC_IN | PF0 | 外部高速晶振输入 (HSE) |
| OSC_OUT | PF1 | 外部高速晶振输出 |
| NRST | NRST | 复位引脚 |
| BOOT0 | PB8-BOOT0 | 启动模式选择 |
| VDD | 多引脚 | 电源 3.3V |
| VSS | 多引脚 | 地 |
| VREF+ | VREF+ | 参考电压 3.3V |

## LED 指示灯

| 引脚 | 功能 | 说明 |
|------|------|------|
| PD14 | LED1 | 用户 LED，高电平亮 |

## 通信接口

### FDCAN2

| 引脚 | 功能 | 说明 |
|------|------|------|
| PB12 | FDCAN2_RX | CAN 接收 |
| PB13 | FDCAN2_TX | CAN 发送 |

### UART4

| 引脚 | 功能 | 说明 |
|------|------|------|
| PC10 | UART4_TX | UART4 发送 |
| PC11 | UART4_RX | UART4 接收 |

### UART5

| 引脚 | 功能 | 说明 |
|------|------|------|
| PC12 | UART5_TX | UART5 发送 |
| PD2 | UART5_RX | UART5 接收 |

## 调试接口

| 引脚 | 功能 | 说明 |
|------|------|------|
| PA13 | SWDIO | SWD 数据 I/O |
| PA14 | SWCLK | SWD 时钟 |

## 原理图功能定义

根据原理图中的信号定义：

| 信号名 | 引脚 | 说明 |
|--------|------|------|
| ADC_VS1-8 | 多通道 | 电压采样输入 |
| ADC_CUR1-12 | 多通道 | 电流采样输入 |
| ADC_TEMP1-2 | PB14, PB15 | 温度采样输入 |
| FDCAN2_RX | PB12 | CAN 接收 |
| FDCAN2_TX | PB13 | CAN 发送 |
| LED1 | PD14 | 状态指示 LED |
